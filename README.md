- 🧠 Learn 1500+ words and phrases
- 📝 Quiz mode 
- 🔎 Search words
- 📦 Offline support using Room

# 🔧 Tech Stack

| Layer          | Technology                         |
|----------------|-------------------------------------|
| Local Storage  | Room Database, DataStore            |
| DI             | Hilt (Dagger-Hilt)                  |
| Language       | Kotlin                              |
| Build System   | Gradle (KTS)                        |
